🌾 AI Agricultural ChatBot - Testing Guide

✅ Features Implemented

1. Position: ChatBot appears at bottom-right corner of the screen
2. Language Support: English and Tamil language toggle
3. Voice Input: Click microphone icon to use voice commands
4. Agricultural Knowledge: Crop info, diseases, schemes, weather

🧪 How to Test

  1. Start the development server: 
    cd frontend
npm run dev
  2. Navigate to Dashboard: Login to your account
  3. Look for green chat icon: Bottom-right corner
  4. Click to open: Chat window opens
  5. Test language toggle: Click "தமிழ்" button
  6. Test voice input: Click microphone icon
  7. Ask questions: Type or speak your queries

💬 Sample Questions to Try

English:

  • "Tell me about rice cultivation"
  • "What are the government schemes?"
  • "How to control crop diseases?"
  • "What is the best weather for wheat?"

Tamil:

  • "நெல் சாகுபடி முறைகள்"
  • "அரசு திட்டங்கள் என்ன?"
  • "பயிர் நோய்களை கட்டுப்படுத்துவது எப்படி?"
