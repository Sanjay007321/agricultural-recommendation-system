# 🌾 AI ChatBot - Quick Start Guide

## 🚀 Get Started in 3 Steps

### Step 1: Start the Server
```bash
cd frontend
npm run dev
```

### Step 2: Login to Your Account
- Open browser (Chrome/Edge recommended)
- Navigate to the application
- Login with your credentials

### Step 3: Open the ChatBot
- Look for **green circular button** at bottom-right corner
- Click to open
- Start chatting!

---

## 🎯 Key Features At a Glance

| Feature | How to Use |
|---------|-----------|
| 💬 **Chat** | Type and press Enter or click send |
| 🎤 **Voice** | Click microphone, speak, auto-transcribes |
| 🌐 **Language** | Click "தமிழ்" or "English" button |
| ❌ **Close** | Click red X button |

## ⚡ Quick Tips

✅ **Best Browser**: Chrome or Edge for full voice support  
✅ **Microphone**: Allow permission when prompted  
✅ **Language**: Switch anytime during conversation  
✅ **Voice**: Speak clearly for best transcription  

❌ **Safari**: Voice input not supported  
❌ **No Mic**: Use keyboard typing instead

## 🔧 Troubleshooting

| Issue | Solution |
|-------|----------|
| No chat icon? | Refresh page, check console |
| Voice not working? | Use Chrome/Edge, allow mic |
| Tamil not showing? | Clear cache, reload |
| Bot not responding? | Check internet connection |
