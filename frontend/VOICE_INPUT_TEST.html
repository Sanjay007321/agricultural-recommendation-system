Troubleshooting Guide

If voice input is not working:

1. Use Chrome or Edge: These browsers have the best speech recognition support
2. Check microphone permissions: Click the lock icon in address bar
3. Enable microphone: Allow microphone access when prompted
4. Check microphone hardware: Ensure your microphone is connected and working
5. HTTPS required: Speech recognition requires secure context (HTTPS or localhost)
6. Update browser: Make sure you're using the latest browser version
